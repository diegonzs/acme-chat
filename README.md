# Acme Chat

Acme Chat is just a simple chat.

## Tools used

- Next JS (Framework)
- styled jsx (Styles)
- Vercel (Deploy)

## Installation

1- Use yarn or npm to install all dependencies.

```bash
yarn install
```

or

```bash
npm install
```

## Run on Local

```bash
yarn dev
```

or

```bash
npm run dev
```

## Demo

You can find a demo in [here](https://acme-chat-psi.vercel.app/) already deployed using vercel.
